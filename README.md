## How to run
### 1. Install dependencies:
- Git
- npm

### 2. Clone this repo
`git clone https://github.com/anovodarskiy/bookmarks.git -b master`
### 3. Install aplication depentensies using npm
`cd bookmarks`
`npm ci`
### 4. Run aplication
`npm run start`
### 5. Run test
`npm run test`
### 6. Run build
`npm run build`
### 7. Run linting
`npm run lint`
